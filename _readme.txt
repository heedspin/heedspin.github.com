
## Install
  gem install jekyll
  brew install python 
  
  gem install less 
  - (for css stuff)
  - [local] $ rake -f _tasks/build.rb

  easy_install Pygments

## Running

  jekyll
  ==> generates html

  jekyll server
  ==> runs webrick
